📖 User Guide

🔹 About

SOLO M Game Shop သည် Mobile Legends: Bang Bang ကစားသူများအတွက် လုံခြုံစိတ်ချရသော အကောင့်စီမံခန့်ခွဲမှုစနစ်တစ်ခုဖြစ်ပါသည်။

🔹 စတင်အသုံးပြုနည်း

  • ၁။ Register လုပ်ပါ - အချက်အလက်များ ဖြည့်သွင်းပြီး အကောင့်အသစ်ဖွင့်ပါ။
  • ၂။ Login ဝင်ပါ - မှတ်ပုံတင်ပြီးသော Email နှင့် Password ဖြင့် ဝင်ရောက်ပါ။
  • ၃။ Dashboard - ပင်မစာမျက်နှာတွင် လက်ကျန်ငွေ၊ Notice များ ကြည့်ရှုနိုင်ပါသည်။
  • ၄။ Top Up - ငွေဖြည့်သွင်းရန် Top Up စာမျက်နှာသို့ သွားရောက်ပါ။
  • ၅။ Buy Code - MLBB Redeem Code များ ဝယ်ယူရန် Buy Code စာမျက်နှာသို့ သွားရောက်ပါ။
  • ၆။ Data Tab - မိမိအကောင့်များ၏ အချက်အလက်များ သိမ်းဆည်းနိုင်ပါသည်။
  • ၇။ Lucky Spin - ကံစမ်းမဲဘူးဖြင့် USD နှင့် MMK ဆုများ ရယူနိုင်ပါသည်။
  • ၈။ Exchange - USD မှ MMK သို့ ငွေလဲလှယ်နိုင်ပါသည်။

🔹 Login နည်းလမ်းများ

  • Local Login - Email နှင့် Password ဖြင့် ဝင်ရောက်ခြင်း
  • Google Login - Google အကောင့်ဖြင့် ဝင်ရောက်ခြင်း
  • Telegram Login - Telegram Bot (@SoloM28_bot) မှတဆင့် ဝင်ရောက်ခြင်း
  • Guest Login - အကောင့်မရှိဘဲ ဝင်ရောက်ကြည့်ရှုခြင်း (အချို့ Features များ ကန့်သတ်ထားသည်)

🔹 လုံခြုံရေး

  • အချက်အလက်အားလုံးကို Encryption ဖြင့် ကာကွယ်ထားပါသည်။
  • Database နှစ်ခုဖြင့် Auto Backup ပြုလုပ်ထားပါသည်။
  • သံသယဖြစ်ဖွယ် လုပ်ဆောင်ချက်များကို 24/7 Monitor လုပ်ထားပါသည်။
