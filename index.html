📖 User Guide - အသုံးပြုနည်းလမ်းညွှန်

🔹 About SOLO M Game Shop

SOLO M Game Shop သည် Mobile Legends: Bang Bang (MLBB) ကစားသူများအတွက် အထူးဖန်တီးထားသော လုံခြုံစိတ်ချရသည့် အကောင့်စီမံခန့်ခွဲမှုစနစ်တစ်ခုဖြစ်ပါသည်။ ဤစနစ်တွင် အောက်ပါဝန်ဆောင်မှုများ ပါဝင်ပါသည်။

🔹 စတင်အသုံးပြုနည်း (အဆင့်ဆင့်)

  • အဆင့် ၁ - Register (အကောင့်ဖွင့်ခြင်း)
    Username, Email, Phone Number, Password တို့ကို ဖြည့်သွင်း၍ အကောင့်အသစ်တစ်ခု ဖွင့်ပါ။ အကောင့်ဖွင့်ပြီးပါက အလိုအလျောက် Admin ထံသို့ အကြောင်းကြားမည်ဖြစ်သည်။
  • အဆင့် ၂ - Login (အကောင့်ဝင်ခြင်း)
    အကောင့်ဖွင့်ပြီးပါက Email နှင့် Password ဖြင့် ဝင်ရောက်နိုင်ပါသည်။ Google, Telegram, Guest အကောင့်များဖြင့်လည်း ဝင်ရောက်နိုင်ပါသည်။
  • အဆင့် ၃ - Dashboard (ပင်မစာမျက်နှာ)
    အကောင့်ဝင်ပြီးပါက Dashboard တွင် မိမိ၏ လက်ကျန်ငွေ (MMK နှင့် USD)၊ အသိပေးချက်များ၊ Video များကို ကြည့်ရှုနိုင်ပါသည်။
  • အဆင့် ၄ - Top Up (ငွေဖြည့်ခြင်း)
    Dashboard မှ Top Up စာမျက်နှာသို့ သွားရောက်၍ Wave Pay, KBZ Pay, AYA Pay တို့မှတဆင့် ငွေဖြည့်သွင်းနိုင်ပါသည်။ ငွေဖြည့်ပြီးပါက ပြေစာ Screenshot ကို Admin ထံ တင်ပြရမည်ဖြစ်သည်။
  • အဆင့် ၅ - Buy Code (Code ဝယ်ယူခြင်း)
    MLBB Redeem Code များကို ဝယ်ယူနိုင်ပါသည်။ Code အမျိုးအစား (Shhh Emote, Golden Border, Lucky Diamond, Magic Durt, Emblem Box) များမှ မိမိနှစ်သက်ရာကို ရွေးချယ်ဝယ်ယူနိုင်ပါသည်။ Code ဝယ်ယူပြီး ၁၀ စက္ကန့်အတွင်း ကူးယူရမည်ဖြစ်သည်။
  • အဆင့် ၆ - Data Management (ဒေတာစီမံခန့်ခွဲမှု)
    မိမိ၏ Gmail, MLBB, TikTok အကောင့်များ၏ အချက်အလက်များကို လုံခြုံစွာ သိမ်းဆည်းနိုင်ပါသည်။
  • အဆင့် ၇ - Lucky Spin (ကံစမ်းမဲဘူး)
    နေ့စဉ် အခမဲ့ ၁ ကြိမ် ကံစမ်းနိုင်ပြီး USD နှင့် MMK ဆုများ ရယူနိုင်ပါသည်။ Premium ဝယ်ယူထားပါက ပိုမိုသော အခွင့်အရေးများ ရရှိမည်ဖြစ်သည်။
  • အဆင့် ၈ - Exchange (ငွေလဲလှယ်ခြင်း)
    Lucky Spin မှ ရရှိသော USD များကို MMK သို့ လဲလှယ်နိုင်ပါသည်။ Exchange Rate မှာ 1 USD = 2,000 Ks (Service Fee 1,000 Ks) ဖြစ်သည်။
  • အဆင့် ၉ - Premium (အဆင့်မြင့်အသုံးပြုသူ)
    Premium ၁လ (Bronze - 8,000 Ks), ၂လ (Silver - 15,000 Ks), ၃လ (Gold - 22,000 Ks) ဝယ်ယူနိုင်ပြီး နေ့စဉ်လှည့်ခွင့်ပိုမိုရရှိခြင်း၊ ဆုရနိုင်ခြေတိုးမြှင့်ခြင်း၊ SUPER Prize ရယူနိုင်ခြင်း စသည့်အကျိုးခံစားခွင့်များ ရရှိမည်ဖြစ်သည်။

🔹 အမေးများသော မေးခွန်းများ (FAQ)

  • မေး: Password မေ့သွားရင် ဘယ်လိုလုပ်ရမလဲ?
    ဖြေ: Admin ကို ဆက်သွယ်၍ အကောင့်ပြန်လည်ရယူနိုင်ပါသည်။
  • မေး: Top Up လုပ်ပြီး ဘယ်လောက်ကြာရင် Balance တက်မလဲ?
    ဖြေ: Admin မှ အတည်ပြုပြီးပါက ချက်ချင်း Balance တက်ပါမည်။
  • မေး: Code ဝယ်ပြီး အသုံးမရရင် ဘယ်လိုလုပ်ရမလဲ?
    ဖြေ: Customer Service မှတဆင့် Screenshot ဖြင့် တိုင်ကြားနိုင်ပါသည်။ စစ်ဆေးအတည်ပြုပြီးပါက ငွေပြန်လည်ပေးလျော်မည်။
  • မေး: Guest Login နဲ့ Registered User ဘာကွာလဲ?
    ဖြေ: Guest Login သည် Top Up, Buy Code, Data Management, Exchange, History စသည့် Feature များကို အသုံးပြုခွင့်မရှိပါ။
