📖 SOLO M Game Shop - အသုံးပြုနည်းလမ်းညွှန်

ဤလမ်းညွှန်သည် SOLO M Game Shop ၏ အင်္ဂါရပ်အားလုံးကို အသေးစိတ်ရှင်းပြထားပါသည်

🔹 SOLO M Game Shop ဆိုတာ ဘာလဲ?

SOLO M Game Shop သည် ဂိမ်းကစားသူများအတွက် အထူးဖန်တီးထားသော ဘက်စုံဝန်ဆောင်မှုစနစ်တစ်ခုဖြစ်ပါသည်။ ငွေဖြည့်သွင်းခြင်း၊ Game Code ဝယ်ယူခြင်း၊ ကံစမ်းမဲဘူး၊ ငွေလဲလှယ်ခြင်း၊ ဒေတာစီမံခန့်ခွဲမှု၊ Chat စနစ် စသည့် ဝန်ဆောင်မှုများစွာ ပါဝင်ပါသည်။

🔹 အကောင့်စနစ်

Local Login - Email နှင့် Password ဖြင့် အကောင့်ဖွင့်၍ ဝင်ရောက်နိုင်ပါသည်။ အကောင့်ဖွင့်ရန် Register Form တွင် Username, Email, Phone, Password တို့ဖြည့်သွင်းရပါမည်။

Google Login - Google အကောင့်ဖြင့် တစ်ချက်နှိပ် ဝင်ရောက်နိုင်ပါသည်။

Telegram Login - Telegram Bot (@SoloM28_bot) မှတဆင့် OTP ကုဒ်ဖြင့် ဝင်ရောက်နိုင်ပါသည်။

Guest Login - အကောင့်မရှိဘဲ ဝင်ရောက်ကြည့်ရှုနိုင်ပါသည်။ (Feature အချို့ ကန့်သတ်ထားသည်)

QR Code Login - Register လုပ်စဉ်က ရရှိသော Recovery QR Code ကို Scan လုပ်၍ အလွယ်တကူ Login ဝင်နိုင်ပါသည်။

🔹 အဓိက ဝန်ဆောင်မှုများ

  • 💰 Top Up (ငွေဖြည့်ခြင်း)
    Wave Pay, A+ Pay တို့မှတဆင့် ငွေဖြည့်သွင်းနိုင်ပါသည်။ ငွေလွှဲပြေစာ Screenshot ကို Admin ထံ တင်ပြရမည်ဖြစ်ပြီး Admin မှ အတည်ပြုပြီးပါက သင့်အကောင့်ထဲသို့ ငွေဖြည့်သွင်းပေးပါမည်။
  • 🛒 Buy Code (Code ဝယ်ယူခြင်း)
    ဂိမ်းဆိုင်ရာ Redeem Code များကို ဝယ်ယူနိုင်ပါသည်။ Code အမျိုးအစားများစွာ ရှိပါသည်။
  • 🎰 Lucky Spin (ကံစမ်းမဲဘူး)
    နေ့စဉ် အခမဲ့ ကံစမ်းနိုင်ပြီး USD နှင့် MMK ဆုများ ရယူနိုင်ပါသည်။ Premium ဝယ်ယူထားပါက ဆုရနိုင်ခြေ ပိုမိုမြင့်မားပါသည်။
  • 💱 Exchange (ငွေလဲလှယ်ခြင်း)
    Lucky Spin မှ ရရှိသော USD များကို MMK သို့ လဲလှယ်နိုင်ပါသည်။
  • 📊 Data Management (ဒေတာစီမံခန့်ခွဲမှု)
    သင့်၏ Gmail, MLBB, TikTok အကောင့်များ၏ အချက်အလက်များကို လုံခြုံစွာ သိမ်းဆည်းနိုင်ပါသည်။
  • 💬 Chat System (စကားပြောစနစ်)
    Admin နှင့် တိုက်ရိုက်စကားပြောနိုင်ပါသည်။ Premium User များ ဓာတ်ပုံနှင့် ဖိုင်များ ပေးပို့နိုင်ပါသည်။
  • 👑 Premium System
    Bronze, Silver, Gold Tier ၃ မျိုးဖြင့် အဆင့်မြင့် ဝန်ဆောင်မှုများ ရယူနိုင်ပါသည်။

🔹 လုံခြုံရေး

  • ✅ အချက်အလက်အားလုံးကို Encryption ဖြင့် ကာကွယ်ထားပါသည်
  • ✅ Database နှစ်ခု (DB1 + DB2) ဖြင့် Auto Backup ပြုလုပ်ထားပါသည်

📱 QR Code System - အသုံးပြုနည်းလမ်းညွှန်

🔹 QR Code Recovery System ဆိုတာ ဘာလဲ?

QR Code Recovery System သည် သင့် အကောင့်အချက်အလက်များ (Email, Password) ကို QR Code ထဲတွင် လုံခြုံစွာ သိမ်းဆည်းထားသည့် စနစ်ဖြစ်ပါသည်။ Password မေ့သွားပါက ဤ QR Code ကို Scan လုပ်၍ အလွယ်တကူ ပြန်လည်ရယူနိုင်ပါသည်။

⚠️ ဤစနစ်ကို မည်သည့် Website Developer မှ မပြုလုပ်ဖူးသေးသော ထူးခြားသော Feature တစ်ခုဖြစ်ပါသည်။

🔹 QR Code ဘယ်လိုရယူရမလဲ?

  1. အဆင့် ၁ - Register လုပ်ပါ
    Register Form တွင် Username, Email, Phone, Password တို့ကို ဖြည့်သွင်းပါ။
  2. အဆင့် ၂ - QR Code ရယူပါ
    Register အောင်မြင်ပါက QR Code Modal ပေါ်လာပါမည်။
  3. အဆင့် ၃ - QR Code သိမ်းပါ
    💾 Save QR Code ခလုတ်ကိုနှိပ်၍ သင့်ဖုန်းထဲတွင် သိမ်းဆည်းထားပါ။

🔹 QR Code ဘယ်လိုသုံးရမလဲ?

📌 နည်းလမ်း ၁ - QR Code Scan (အကြံပြု)

  1. Login Page မှ "Scan QR Code to Login" ခလုတ်ကိုနှိပ်ပါ
  2. သင့်ဖုန်းထဲရှိ QR Code ကို ကင်မရာဖြင့် Scan လုပ်ပါ
  3. Email နှင့် Password သည် အလိုလျောက် ဖြည့်သွင်းပေးပါမည်
  4. Login ခလုတ်ကိုနှိပ်၍ ဝင်ရောက်ပါ

📌 နည်းလမ်း ၂ - QR Code Upload

  1. "Scan QR Code to Login" ခလုတ်ကိုနှိပ်ပါ
  2. သင့်ဖုန်းထဲမှ QR Code ပုံကို Upload လုပ်ပါ
  3. စနစ်မှ QR Code ကို ဖတ်ပေးပါမည်

⚠️ အရေးကြီး မှတ်ချက်များ

  • QR Code ကို လုံခြုံစွာ သိမ်းဆည်းထားပါ။ တစ်ခြားသူများ ရရှိပါက သင့်အကောင့်ကို ဝင်ရောက်နိုင်ပါသည်။
  • Password ပြောင်းလဲပါက QR Code အသစ် ပြန်လည်ရယူရပါမည်။
  • QR Code ပျောက်ဆုံးပါက Admin ထံ ဆက်သွယ်၍ အကောင့်ပြန်လည်ရယူနိုင်ပါသည်။
  • QR Code သည် Register လုပ်တိုင်း အသစ်ထုတ်ပေးပါသည်။

💡 အကြံပြုချက် - QR Code ကို Screenshot ရိုက်ထားပြီး လုံခြုံသောနေရာတွင် သိမ်းဆည်းထားရန် အကြံပြုပါသည်။
