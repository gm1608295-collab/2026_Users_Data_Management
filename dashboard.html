Data များအကြောင်း

📧 Gmail Data - သင်၏ Gmail အကောင့်အချက်အလက်များကို ဤနေရာတွင် သိမ်းဆည်းနိုင်ပါသည်။ Email, Phone, Password, DOB, Country/Region တို့ကို ဖြည့်သွင်းနိုင်ပြီး Email နှင့် Phone တို့ကို တစ်ခုထက်ပို၍ ထည့်သွင်းနိုင်ပါသည်။


🎮 MLBB Data - သင်၏ Mobile Legends အကောင့်အချက်အလက်များ (In-Game Name, ID, Server ID) တို့ကို ဤနေရာတွင် သိမ်းဆည်းနိုင်ပါသည်။


🎵 TikTok Data - သင်၏ TikTok အကောင့်အချက်အလက်များကို ဤနေရာတွင် သိမ်းဆည်းနိုင်ပါသည်။


🔒 Generator Password (အများသုံး) - အောက်ပါ Password များသည် အများသုံး Public Password များဖြစ်ပါသည်။

🔑 ကိုယ်ပိုင် Password ပြောင်းလဲခြင်း - အထက်ပါ Password များသည် အများသုံးဖြစ်ပါသည်။ မိမိတစ်ဦးတည်းသာ သုံးစွဲလိုပါက Menu Tab ထဲရှိ Password တွင် လက်ရှိ Password အဟောင်းကိုထည့်၍ မိမိစိတ်ကြိုက် Password အသစ်တစ်ခုကို ပြောင်းလဲနိုင်ပါသည်။


📋 History - မိမိသိမ်းဆည်းထားသော အချက်အလက်များကို History Tab တွင် ပြန်လည်ကြည့်ရှုနိုင်ပြီး View/Edit မှတစ်ဆင့် ပြန်လည်ပြင်ဆင်နိုင်ပါသည်။


📞 Contact - တစ်စုံတစ်ရာ ပြဿနာရှိပါက Contact Tab မှတစ်ဆင့် Gmail, Phone, Telegram, TikTok တို့ဖြင့် ဆက်သွယ်နိုင်ပါသည်။ EmailJS စနစ်ဖြင့်လည်း စာပို့နိုင်ပါသည်။


🔔 Notice - Admin ဘက်မှ စနစ်အသစ်များ၊ Version အသစ်များ ထွက်ရှိပါက Notice ဖြင့် တစ်ရက်ကြိုတင် အကြောင်းကြားပေးပါသည်။
