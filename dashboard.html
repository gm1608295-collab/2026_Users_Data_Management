U ID အကြောင်း

🔹 U ID ဆိုတာဘာလဲ?
U ID သည် သင့်အကောင့်၏ သီးသန့်နံပါတ်ဖြစ်ပြီး အကောင့်တစ်ခုချင်းစီအတွက် သီးသန့်သတ်မှတ်ထားပါသည်။
🔹 U ID ကို ဘယ်မှာသုံးလဲ?
Top Up System တွင် ငွေဖြည့်သွင်းရာတွင် User ID အကွက်ထဲတွင် ထည့်သွင်းရန် လိုအပ်ပါသည်။ ID မှန်မှသာ ငွေဖြည့်သွင်းနိုင်ပါမည်။
🔹 ID ကို ဘယ်လိုကူးယူမလဲ?
U ID ဘေးရှိ Copy icon ကိုနှိပ်ပြီး အလွယ်တကူကူးယူနိုင်ပါသည်။
🔹 ID ဘာကြောင့်အရေးပါလဲ?
ငွေဖြည့်သွင်းရာတွင် ID မှားယွင်းပါက ငွေဖြည့်သွင်းနိုင်မည်မဟုတ်ပါ။ ထို့ကြောင့် ID မှန်ကန်ရန် အလွန်အရေးကြီးပါသည်။
